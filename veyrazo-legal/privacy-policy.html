Veyrazo Privacy Policy

Veyrazo is a home-friendly fitness companion that helps users create an account, choose fitness preferences, follow routines, track progress, complete daily quests, and grow an in-app tree based on consistent activity.

This policy describes the data Veyrazo uses for account security, app personalization, progress tracking, social features, and cloud sync.

Last updated: June 30, 2026

Account Data

Veyrazo may collect or store the following when needed for the selected feature:

  • Username
  • Email address
  • Display name
  • Birthday
  • Profile photo when uploaded by the user
  • Authentication status, session status, email verification status, and account security preferences

This data is used to create and secure the account, support sign-in by email or username, restore account data across devices, support account recovery and two-factor security, and display safe public profile details for friends, friend challenges, and leaderboard features.

Fitness Profile Data

Veyrazo may collect or store the following when needed for the selected feature:

  • Main fitness goal
  • Fitness level
  • Activity level
  • Age, height, and weight when provided
  • Workout days, preferred workout duration, routine length, and low-impact preferences
  • Limitation or injury notes entered by the user

This data is used to personalize workout recommendations, generate routine plans, avoid unsafe intensity jumps, and improve safety guidance and low-impact alternatives.

Progress And Activity Data

Veyrazo may collect or store the following when needed for the selected feature:

  • Weight logs and hydration logs
  • Step totals and workout completions
  • Exercises completed, skipped exercises, sets, reps, duration, and estimated calories
  • Daily quest completions and recovery task completions
  • Outdoor activity summaries such as activity type, duration, distance, pace, speed, estimated calories, step estimate, elevation range, target type, and route point count
  • Private route points when outdoor tracking is enabled
  • Tree progress and achievement unlock summaries

This data is used to show dashboard and progress summaries, restore progress after sign-in, generate tree growth and achievement summaries, and support friend challenges and leaderboard-safe summaries without exposing private health details.

Health Connect

When available and enabled by the user, Veyrazo requests Health Connect read access for steps, distance, active calories burned, exercise sessions, and compatible heart-rate data when supported by the connected wearable or health source.

This data is used to automatically update step totals, improve progress summaries, and reduce manual logging. Users can deny Health Connect access and continue using manual tracking.

Location Data

When the user starts an outdoor walk, run, hike, cycling, or supported cardio tracking session, Veyrazo requests location permission to estimate route, distance, pace, speed, time, elevation, and estimated calories.

Precise route data is private by default. Public sharing, friends, and leaderboards must use sanitized summaries and must not expose exact home or work routes.

Notifications

Veyrazo requests notification permission for workout reminders, safety prompts, achievement alerts, and weekly summaries. Users can disable reminder-style notifications from settings.

Cloud Storage And Sync

Veyrazo uses Firebase Authentication, Cloud Firestore, Firebase Storage, and Cloud Functions for identity, account data, onboarding, routine, progress, tree, and achievement support. Some data is stored locally on the device first, then synced when the user is signed in.

Public Data

The app may create public profile summaries with safe fields only, including display name, username, profile photo URL, tree stage, and total achievement count.

Public profile summaries must not include email address, birthday, exact weight, limitation notes, or precise route and location data.

Data We Do Not Store

  • Passwords or password hashes in Firestore
  • Raw two-factor codes
  • Full contact lists
  • Precise route and location data in public documents
  • Unredacted support or debug logs

Firebase Authentication manages password credentials.

Data Sharing

Veyrazo does not sell personal data.

Veyrazo may use safe public summaries for friend requests, friend challenges, leaderboards, invites, ads, and donations. These summaries are limited to profile and activity information needed to operate those features and do not include private account data, exact weight, limitation notes, or precise route points.

User Controls

  • Sign out
  • Change account details
  • Reset local progress
  • Delete the account
  • Disable notification reminders
  • Control Health Connect access through Android system settings
  • Control location permission through Android system settings

Data Retention

Local data remains on the device until the user clears app data, resets local progress, signs out where local reset is triggered, or deletes the account.

Cloud data remains until the user deletes the account or Sloth Bloom removes it as part of normal security, maintenance, or legal compliance processes.

Children

Veyrazo is not intended for children under 13. Account creation requires a birthday that indicates the account holder is at least 13.

Safety

Veyrazo provides general fitness and wellness guidance. It is not medical advice, diagnosis, treatment, physical therapy, or emergency care.

Contact

For support or privacy questions, use the Help and Legal sections in the Veyrazo app. A public support email or web form should be connected before full production launch.
